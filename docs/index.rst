Installation
++++++++++++
.. _installation:

How to install the myanimelistpy library.

.. code:: sh

   pip install myanimelistpy

or

.. code:: sh

   # Linux/macOS
   python3 -m pip install -U myanimelistpy

   # Windows
   py -m pip install -U myanimelistpy
