Installation
------------

How to install the myanimelistpy package.

.. code:: sh

   pip install myanimelistpy

or

.. code:: sh

   # Linux/macOS
   python3 -m pip install -U myanimelistpy

   # Windows
   py -m pip install -U myanimelistpy
